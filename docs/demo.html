The grid below is a live, interactive MVCGrid — sorting, paging, filtering and CSV export all work. What makes it unusual: there is no server. The whole MVCGrid engine is running in your browser via .NET WebAssembly (the MVCGrid.Wasm host feeding the same MVCGrid.js client script the server-side hosts use), so it can be served as static files straight from GitHub Pages.

The first load fetches the .NET runtime (a few MB, then browser-cached). Source: MVCGrid.WasmExample.
